## Quick Installation

### Claude Code Marketplace

```bash
/plugin marketplace add ringprotocol/ring-ai
```

### Install Individual Plugins

```bash
/plugin install ring-trading       # Swap integration
/plugin install ring-driver        # Swap & liquidity planning
```
