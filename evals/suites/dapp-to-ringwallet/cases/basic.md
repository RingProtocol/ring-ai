# Basic Ring Wallet DApp Integration

Explain how to integrate a DApp into Ring Wallet for a team that has never done it.

## Context

- The DApp is a web app that already uses wagmi and RainbowKit
- The team needs the exact integration steps and testing flow
- The wallet host is <https://wallet.ring.exchange>

## Requirements

1. Include the registration info required to receive an API key
2. Explain the two SDK loading options and recommend one
3. Show the script tag placement in the HTML head
4. Explain CSP requirements for the SDK
5. Explain what the SDK injects, including EIP-1193 and EIP-6963 behavior
6. Explain iframe embedding requirements and X-Frame-Options fixes
7. Provide the test URL format using the testdapp query parameter

## Expected Output

Clear, ordered steps that a DApp team can follow to integrate and test within Ring Wallet.
